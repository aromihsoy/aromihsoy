Backend-разработчик с опытом в IT-инфраструктуре - системном администрировании, сетях и поддержке.

Сейчас сосредоточен на чистой архитектуре, проектировании реляционных баз данных и построении стабильных серверных приложений.

### 🌱 Чем я занимаюсь

- Разрабатываю асинхронные веб-сервисы на **Python** (FastAPI).
- Проектирую схему БД и работаю с **PostgreSQL** через **SQLAlchemy**.
- Строю event-driven системы на **RabbitMQ**: ручная AMQP-топология,
  transactional outbox, идемпотентные консьюмеры, DLQ и retry с backoff.
- Настраиваю кэширование и Pub/Sub на **Redis**, фоновые задачи через **Celery**.
- Упаковываю приложения в **Docker** для воспроизводимого развёртывания.
- Тестирую и документирую API в **Postman**.

### 🚀 Что осваиваю

- Изучаю **Django** для разработки полноценных веб-приложений.
- Следующим планирую осваивать **Go** для микросервисной разработки.
